# Hellshake-Yano Text形式辞書ファイル
# シンプルなテキスト形式でユーザー定義辞書を作成
#
# 形式:
# - 通常の行: カスタム単語
# - !で開始: 分割禁止ワード
# - =を含む: 結合ルール
# - @で開始: ヒントパターン (優先度:パターン:位置)
# - #で開始: コメント

# カスタム単語
機械学習
深層学習
人工知能
ニューラルネットワーク
アルゴリズム

# 分割禁止ワード（!で開始）
!HelloWorld
!getElementById
!addEventListener
!TypeScript
!JavaScript

# 結合ルール（単語=戦略）
の=always
を=always
に=always
が=always
で=always

# ヒントパターン（@優先度:パターン:位置）
@100:^-\s*\[\s*\]\s*(.)：capture:1
@95:^\d+\.\s*(.)：capture:1
@90:^#+\s*(.)：capture:1
@80:「([^」])：capture:1
