# Instructions for Testing (テスト手順)

1. このファイルを開く
2. 各設定ファイルを :source コマンドで読み込む
3. 'w', 'b', 'e' キーで単語移動を試す
4. 各行での単語境界を validation-results.md に記録する
5. 主観的な評価（スムーズさ、精度）を記録する
