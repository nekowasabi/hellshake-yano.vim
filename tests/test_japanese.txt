# 日本語ヒント表示テスト用ファイル

## 1文字の日本語単語
私は本を読む。今日は良い天気だ。

## ひらがなのテスト
これはよいものだ。あれもそうですね。

## カタカナのテスト
カタカナテスト。コードレビュー。プログラミング。

## 漢字のテスト
今日明日明後日。春夏秋冬。東西南北。

## 混在テキスト
私はProgrammerです。今日はgood dayですね。
これはtest codeです。日本語とEnglishが混在。

## 長い日本語文章
日本語の文章を書くことは、英語とは異なる難しさがあります。
特に、単語の境界が明確でないため、ヒント表示の実装には工夫が必要です。

## プログラミング用語の混在
関数functionを定義defineする。変数variableを宣言declareする。
配列arrayを初期化initializeする。オブジェクトobjectを作成createする。

## 記号を含む文章
これは「引用」です。それは『書名』です。
メールアドレス：test@example.com
URLアドレス：https://example.com

## 数字を含む文章
2025年1月1日。第3章第2節。100個のアイテム。

# Vim設定例（.vimrcまたはinit.vimに追加）

```vim
let g:hellshake_yano = {
  \ 'use_japanese': v:true,
  \ 'word_detection_strategy': 'hybrid',
  \ 'enable_tinysegmenter': v:true,
  \ 'use_improved_detection': v:true,
  \ 'single_char_keys': ['A', 'S', 'D', 'F', 'G', 'H', 'J', 'K', 'L', ';'],
  \ 'multi_char_keys': ['Q', 'W', 'E', 'R', 'T', 'Y', 'U', 'I', 'O', 'P'],
  \ 'motion_count': 3,
  \ 'motion_timeout': 2000
\ }
```

## 動作確認方法

1. 上記の設定を.vimrcまたはinit.vimに追加
2. Neovimを再起動
3. このファイルを開く: `:e test_japanese.txt`
4. hjklキーを3回押してヒントを表示
5. 日本語の各文字にヒントが表示されることを確認

## 期待される動作

- 「私」「本」「今」「日」などの1文字漢字にヒントが表示される
- 「あ」「い」「う」などのひらがなにヒントが表示される
- 「カ」「タ」「ナ」などのカタカナにヒントが表示される
- 「Programmer」「good」「day」などの英単語にもヒントが表示される
- 日本語文字は1文字ごとに個別のヒントが割り当てられる
